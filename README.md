### 💻 Projeto

O projeto trata-se de uma versão web do classico Jogo da Cobrinha.
Teclas de Movimentação: 
- <strong>W</strong>  (cima)
- <strong>D</strong>  (direita)
- <strong>S</strong>  (baixo)
- <strong>A</strong>  (esquerda)
